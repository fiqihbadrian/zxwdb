# Changelog

All notable changes to ZxwDB will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial public release
- Comprehensive documentation and README
- Contributing guidelines
- Security policy
- Issue templates for bug reports and feature requests

## [1.0.0] - 2026-06-10

### Added
- Visual database designer with drag-and-drop interface
- ERD (Entity Relationship Diagram) visualization
- LRS (Logical Relational Schema) detailed view
- Connect to local/remote MySQL/MariaDB databases
- Import existing database schemas (reverse engineering)
- Export SQL scripts and generate migrations
- Real-time visual design to SQL conversion
- Canvas interface with zoom and pan
- Table and column management
- Foreign key relationship visualization
- Data browser for viewing table contents
- SQL query runner with syntax support
- Database connection manager
- Multi-database support
- Auto-save functionality
- Keyboard shortcuts
- Responsive UI with TailwindCSS

### Frontend Technologies
- React 18 with TypeScript
- ReactFlow for canvas visualization
- Zustand for state management
- Axios for API communication
- Vite for fast development

### Backend Technologies
- Node.js with Express
- mysql2 for database connectivity
- CORS support for local development
